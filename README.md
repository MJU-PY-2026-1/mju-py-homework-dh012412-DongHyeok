# 🐍 [2026-01] 나만의 파이썬 소프트웨어 개발 프로젝트

## 1. 시나리오 제목
* 선호도에 따른 장르별 음악 추천 시스템
<br>


## 2. 시나리오 (5~10줄)
* 이 프로그램은 사용자가 선호하는 음악장르별 점수를 입력받아 선호하는 음악을 확인하고, 그 장르에 맞는 음악을 추천해주는 **장르별 음악 추천 시스템** 입니다.
* 사용자는 프로그램이 입력하라는 여러 장르별 점수를 입력합니다. 장르는 [힙합, 알앤비, 발라드, 팝, 인디] 총 다섯 장르에대해 선호 점수를 입력받습니다. 점수를 바탕으로 총점을 구하고, 평균을 구하여 평균보다 점수가 높은 장르에 대해서는 그 장르 내에서 유명하지 않은 음악을 추천해주고, 평균보다 점수가 낮은 장르에 대해서는 그 장르 내에서 유명한 음악을 추천해줍니다. 유명하지 않은 음악/ 유명한 음악의 리스트는 프로그램에 저장해둔 다섯곡에서 무작위로 한 곡을 골라서 출력해주고, 장르별 추천 음악을 모두 들어보았는지 입력받은 뒤, 모두 들어보았다면 '장르 올킬상' 을 제공합니다. 들어보지 않은 장르에 대해서는 그 장르내에서 추천된 음악을 제외하고 다른 음악을 다시 추천하여 들어보았는지 다시 입력받습니다. '장르 올킬상' 을 받은 사용자에 한해서, 추천 음악을 추가할 수 있는 기회가 한 번 주어집니다. 최종적으로 추천된 장르별 한 곡 즉, 음악 다섯 곡과 추가된 음악 한 곡 총 여섯 곡을 출력하여 하나의 플레이리스트를 만들어주는 시스템을 완성할 계획입니다.

<br>


## 3. 예상 기능 및 메뉴 (최소 5개)
(예시)
1. 사용자 등록
2. 장르별 선호 점수 입력
3. 장르별 음악 추천
4. 장르 올킬상 지급 판별
5. 플레이리스트 출력
6. 시스템 종료

## 🟩 [2차 과제: V1.0] 입출력 + 리스트 + 조건문 
### **✨2차 과제 내용:**
  
  * 이번 과제는 사용자 등록 및 선호도 확인 시스템 구현으로, 사용자로부터 여러 음악 장르에대한 선호도를 입력받아, 평균에 따른 장르별 음악을 추천해주는 시스템을 구축합니다. if-else 조건문을 통해 점수와 평균을 비교하여 random.choice() 로 랜덤하게 장르별로 한 곡을 추천해줍니다. 추천 음악들을 모두 들어보았는지 if-elif-else 을 활용해 확인합니다.
  * input() 을 통해 이름을 입력받고, for 을 통해 반복적으로 장르에 대한 선호도와 복합대입연산자를 활용해 총점을 구해 평균을 출력합니다.
  * 각각의 장르에 대한 선호도 점수가 평균보다 높으면 유명하지 않은 음악을 추천하고, 평균보다 낮으면 유명한 음악을 추천합니다.
  * if-elif-else 를 통해 추천음악을 모두 들어보았는지 확인받습니다.
    
    
### **🤖 AI 파트너십 과정**
(예시) 
1. **내용 1: 효율적인 코드 작성을 위한 자문**
    * **프롬프트 요약:** "random.choice([]) 처럼 함수 사이에 배열을 넣으니까 가독성이 떨어지는데, 어떻게 하면 더 효율적으로 작성할 수 있을까?"
    * **적용 내용:** AI가 제안한 유명한 음악과 유명하지 않은 음악에 대한 변수를 분리하고, 맨 앞에 리스트를 배치하며 효율적이고 가독성이 뛰어난 코드를 작성할 수 있음.

2. **내용 2: 유효하지 않는 값 입력 작성 자문**
    * **프롬프트 요약:** "유효하지 않은 선호도 점수를 입력했을 때 어떤 식으로 프로그램을 작성해야 할까?"
    * **적용 내용:** AI 는 while 을 추천했지만, 아직 배우지 않았으므로 for과 continue를 이용하여 유효하지 않은 선호도 점수를 필터링함.
    
### **🛠️ Troubleshooting & 기술 회고:**

1. **문제 1: random.choice() 함수 실행이 안되는 문제 (TypeError)**
    * **원인:** random.choice() 에 괄호를 반복적으로 빠뜨려 TypeError 가 발생.
    * **해결:** random.choice() 도 결국 print() 나 input() 과 같은 하나의 함수라는 것을 이해하고 수정함.
